#### Install

```bash
yarn add @0x/asset-swapper
```

#### Import

```javascript
import { SwapQuoter, SwapQuoteConsumer } from '@0x/asset-swapper';
```

or

```javascript
var SwapQuoter = require('@0x/asset-swapper').SwapQuoter;
var SwapQuoteConsumer = require('@0x/asset-swapper').SwapQuoteConsumer;
```
